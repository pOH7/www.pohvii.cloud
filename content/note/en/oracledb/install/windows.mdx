## Install Oracle Instant Client on Windows

**Note:** Only required for thick mode. Thin mode (default) needs no driver installation.

Download "Basic Package": https://www.oracle.com/database/technologies/instant-client/winx64-64-downloads.html

Extract to a directory (e.g., `C:\oracle\instantclient_21_13`)

Add to PATH in Environment Variables (User or System)

```bash
pip install oracledb
```
